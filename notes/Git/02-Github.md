## GitHub

GitHub ist eine Online-Plattform, auf der Remote-Git-Repositories gespeichert, geteilt und gemeinsam bearbeitet werden können. Mit GitHub kann dieselbe Codebasis von vielen Mitarbeitenden gemeinsam genutzt und bearbeitet werden. Viele Repositories sind Open Source, sodass man den Code einsehen, eine Kopie erstellen, ihn verändern oder in eigenen Projekten verwenden kann.

> 💡 Tipp: Schau dir diese umfangreiche [Liste von GitHub-Repositories](https://github.com/pawelborkar/awesome-repos) an und entdecke, was es dort gibt.

GitHub ist gleichzeitig ein soziales Netzwerk für Entwickler und Unternehmen. Das eigene GitHub-Profil wird ein wertvolles öffentliches Aushängeschild für die zukünftige Karriere sein. Über GitHub kann man mit vielen Open-Source-Projekten, Entwicklern und sogar Unternehmen in Kontakt treten.

> 💡 Auch wenn GitHub die beliebteste Online-Git-Plattform ist, ist sie bei weitem nicht die einzige. Es gibt mehrere Alternativen, z. B. GitLab oder Bitbucket.
