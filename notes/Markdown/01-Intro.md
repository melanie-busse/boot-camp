# Markdown

Die Markdown-Syntax ermöglicht das Schreiben von formatiertem Text (Überschriften, Blockzitate, Listen usw.), der als reiner Text gespeichert werden kann. Sie wird von Tools und Webseiten wie GitHub oder Slack verwendet. Dabei werden bestimmte Zeichen genutzt, um Textteile auf eine bestimmte Weise zu formatieren.

---

## Markdown-Beispiele

| Element | Markdown-Syntax |
|---|---|
| Überschrift Ebene 1 | `# Überschrift Ebene 1` |
| Überschrift Ebene 2 | `## Überschrift Ebene 2` |
| Überschrift Ebene 5 | `##### Überschrift Ebene 5` |
| Listenelement | `- Listenelement` |
| Checkbox (offen) | `[ ] Checkbox` |
| Checkbox (erledigt) | `[x] Checkbox` |
| Fetter Text | `**fetter Text**` |
| Kursiver Text | `_kursiver Text_` |
| Link | `[Linktext](https://www.example.com)` |
| Bild | `![Bildbeschreibung](URL zum Bild)` |
| Blockzitat | `> Blockzitat` |
| Trennlinie | `---` |
| Inline-Codeblock | `` `Inline-Codeblock` `` |
| Codeblock | ` ``` Codeblock ``` ` |

Weitere Markdown-Syntax findest du im [Markdown Cheatsheet](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet).
