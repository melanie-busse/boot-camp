# Überblick

Next.js ist ein Fullstack-Framework, das React als Rendering-Engine nutzt. Daher ist ein grundlegendes Verständnis von React für dieses Modul entscheidend. Diese Einheit ist vollständig optional und behandelt angesichts der verfügbaren Zeit nur das Wesentliche. Nutze sie als Auffrischung, oder überspringe sie, wenn du dich mit React bereits sicher fühlst.
